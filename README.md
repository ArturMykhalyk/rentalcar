# RentalCar

A simple web application for renting cars online.

## Description

**RentalCar** is an easy-to-use platform that allows users to search, view, and rent cars from different locations.  
It connects customers with available vehicles and provides detailed information about each car, including price, availability, and specifications.

## Main Features

- Search cars by price, model, or body type
- View car details (price, brand, transmission, fuel type, etc.)
- Book a car online
- Manage bookings through a user-friendly dashboard
- Design desktop devices

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ArturMykhalyk/rentalcar
   ```
2. Go to the project directory:

cd rentalcar

3. Install dependencies:

   npm install

4. Create a .env file in the root directory and add your environment variables:

NEXT_PUBLIC_API_URL="..."

5. Run the development server:

npm run dev

Usage

Open your browser and go to http://localhost:3000

Search for available cars

Select a car and proceed with booking
